# README (oCMS)

## Summary

Top-level project README for oCMS. Describes features across content management, taxonomy, forms, themes, modules, REST API, SEO, administration, multi-language, webhooks, import/export, and performance. Documents prerequisites (Go 1.26+, Node.js, sqlc, templ, goose, Dart Sass, libvips), installation, Docker setup, the full `OCMS_*` environment variable reference (55+ variables, including production-gate `OCMS_REQUIRE_*` flags), the `make` command surface, REST API endpoints, theme and module developer guides, repository layout, and the technology stack. Licensed under GPL-3.0-or-later.

Current version at time of ingest: **0.18.1** (2026-04-14). Tech stack: Go 1.26+, SQLite + goose, sqlc-generated queries, templ templates, HTMX + Alpine.js, TinyMCE 8 editor, Argon2id password hashing, SCS sessions, optional Redis caching, libvips for image processing.
